## Project Notes

1. Clip Manager - thumbnail grid, tap to preview modal, drag to reorder, delete segments
2. Text Overlay - pro_video_editor image layers (render Flutter text to PNG, overlay)
3. Sound Picker - ~200 bundled classic Vine sounds, scrollable list with search

- pro_video_editor for text overlay (native, no FFmpeg font issues)
