## Camera Default Aspect Ratio Change (2025-11-13)

**Changed default from square (1:1) to vertical (9:16)**

### Files modified:
1. **`lib/services/vine_recording_controller.dart:931`**
   - Changed: `model.AspectRatio _aspectRatio = model.AspectRatio.square;`
   - To: `model.AspectRatio _aspectRatio = model.AspectRatio.vertical;`
   - Added comment: "Default to 9:16 vertical"

2. **`lib/models/aspect_ratio.dart:5-8`**
   - Updated enum comments to reflect vertical is now default
   - Changed: "square, // 1:1 (default, classic Vine)"
   - To: "square, // 1:1 (classic Vine)" and "vertical, // 9:16 (default, modern vertical video)"

### Why:
- Modern mobile video standards use 9:16 vertical
- Better fits mobile screens in natural orientation
- Classic Vine square format still available via UI toggle

### Impact:
- New recordings will default to 9:16 instead of 1:1
- Users can still switch to square via aspect ratio button
- Existing recordings unaffected
